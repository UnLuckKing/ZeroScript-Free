# ZeroScript Golden Templates

Golden Templates are deterministic, tested starting points used by the Prototype Accelerator. They prevent an AI from rebuilding common Roblox systems from scratch.

## Available

- `rng/` — server-authoritative aura RNG prototype with saving, pity, luck upgrades, inventory, equip flow, responsive UI and a compact lobby shell.

## Planned

- simulator
- clicker
- tycoon
- obby
- pet collection

Templates are installed by the extension in a bounded number of Studio tool calls. AI is optional and is used only for theme-specific polish or launch-day verification.
