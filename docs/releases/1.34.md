# ZeroScript One 1.34 — Prototype Accelerator

## Faster creation

- New **15 dk Prototip** mode installs a Golden RNG Template directly instead of asking an AI to rebuild common systems.
- Prototype installation is bounded to two Studio tool calls: install and verification.
- New **1 Günlük Yayın** mode installs the same reliable base, then allows one AI owner to polish and test it.
- Config generation selects title, theme, UI preset, map shell, aura names and progression values from the user's idea.
- The first Golden Template includes server-authoritative roll, pity, luck upgrades, inventory, equip, DataStore saving, responsive UI and a compact lobby.

## Simpler interface

- The desktop workspace now exposes only three clear modes: 15-minute Prototype, Launch Day and Custom Work.
- Prototype mode does not require an AI tab.
- Live progress shows template configuration, installation and verification rather than an estimated multi-agent plan.

## Repository cleanup

- Historical release notes were consolidated under `docs/releases/`.
- Reusable game foundations live under `templates/`.
- Runtime extension code remains under `zeroscript-extension/`; Studio plugin code remains under `roblox-plugin/`.

## Limits

- Version 1.34 ships one Golden Template: RNG/Aura.
- Live Roblox Studio playtesting is still required before publishing.
- Launch Day does not publish the place automatically or invent product/gamepass IDs.
