# ZeroScript One 1.35 — ChatGPT Max

## Added

- ChatGPT-first provider priority for AI-owned Roblox Studio work.
- Visible ChatGPT model and reasoning-level detection in the browser panel.
- **Gücü yükselt** action that selects the strongest visibly available enabled reasoning option.
- Compact Project Capsule containing project identity, Project Genome counts, known systems, recent changed paths, Output signals and verified reports.
- One-pass ChatGPT execution contract: inspect, implement, self-review, playtest, inspect Output, fix and finish.
- Same-pass visual checks for desktop/mobile UI and changed buttons.
- Same-pass server-authority checks for DataStore, purchases, rewards and Remotes.
- Bounded inactivity recovery: at most two continuation nudges when ChatGPT has not used Studio tools or produced evidence.
- ChatGPT Max status in extension/team status payloads.

## Removed from the normal AI workflow

- Separate Analyst task
- Separate Reviewer task
- Separate UI or Map agent handoff
- Separate QA task for ordinary workbench requests

Golden Template prototype mode remains deterministic and does not require ChatGPT.

## Limits

ZeroScript can detect and select only options that are visible and enabled in the current ChatGPT account. It cannot unlock Pro or reasoning levels that the account or rollout does not provide. Live ChatGPT DOM behavior and Roblox Studio tool execution still require Windows/Chrome/Studio testing.
