# ZeroScript 1.31 — Easy Mode

## Beginner-first modern Hub

- New **Kolay Mod** opens by default and hides technical tabs until the user chooses **Gelişmiş araçları göster**.
- One clean screen shows readiness, the recommended next action, active-task progress, connection quality and four one-click improvement workflows.
- Plain Turkish status messages replace raw technical details in the normal workflow.
- The modern dark theme was tightened for better hierarchy, spacing, input readability and smaller advanced tabs.

## One-click game builder

The user writes a normal game idea and chooses:

- game type: RNG/Aura, Simulator, Clicker/Incremental, Tycoon, Obby, Pet Collection or Custom
- target: Fast Prototype, Publishable Game or Premium Quality
- device priority: Mobile, Desktop + Mobile or Desktop

ZeroScript creates a persistent dependent queue rather than one oversized prompt. Stages include:

1. project understanding and Project Genome
2. secure foundation and core gameplay loop
3. progression, economy and persistence
4. professional UI and onboarding
5. retention and fair monetization for publishable targets
6. quality, performance, security and release proof
7. content depth and delight for premium targets
8. independent quality jury for premium targets

Each stage must finish before the next stage can start.

## One-click jobs

- **Oyunu düzelt:** diagnose connection, project, Output and verified system failures
- **UI'yi profesyonelleştir:** modern responsive UI, button tests and before/after evidence
- **Oyunu eğlenceli yap:** improve the verified weakest part of the main loop and progression
- **Yayına hazırla:** security, data, purchase, performance, mobile and release checks

## Smart next action

The Easy Mode dashboard detects the current blocker and offers one relevant button:

- start ZeroScript services
- pair the extension
- repair Studio MCP
- open a model
- show the active task
- repair new Output errors
- write an idea
- build the game plan

## Safety and quality

- Game blueprints use existing task queue dependencies, checkpoints, Project Genome, Memory Vault, Design DNA, Shadow Guard and Proof Engine.
- Creative work is still performed by connected AI providers; the local system only plans, constrains, sequences and verifies it.
- Premium and publishable targets use stronger quality modes and separate proof stages.
- The system never invents gamepass/product IDs and does not silently publish the experience.

## Validation limits

The repository includes JavaScript syntax checks, Easy Mode blueprint tests, Python compile checks and existing Superior Engine/Memory Vault tests. Real Windows layout, Chrome providers, Roblox Studio playtests and long multi-stage game builds still require live testing.
